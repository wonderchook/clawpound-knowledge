# Changelog

## 0.1.0

Initial release. Adapted from [EveryInc/compound-knowledge-plugin](https://github.com/EveryInc/compound-knowledge-plugin) for OpenClaw.

### Features

- Six knowledge work skills: brainstorm, plan, confidence, review, work, compound
- Five research/review agents for parallel analysis
- Three opt-in tools for memory-integrated knowledge storage
- Hooks for automatic knowledge context injection and compound suggestions
- Knowledge entries stored in `memory/knowledge/` for automatic `memory_search` indexing
